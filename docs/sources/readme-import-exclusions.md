# Import Exclusions

Venues and data entries we have intentionally decided **not** to import, with the reason why. Check this list before adding from any source that might overlap with these.

---

## Policy: Inclusivity Requirement for Camping/Lodging

We only import campgrounds and lodging that are welcoming to all LGBTQ+ travelers. Venues that explicitly restrict access to a single gender or orientation subset (e.g., gay men only, women only, lesbians only) are excluded unless they make it clear that trans and non-binary guests of all identities are also welcome.
